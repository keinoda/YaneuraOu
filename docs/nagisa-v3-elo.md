## 結果の見方

- **勝率**は、引き分けを除いた対局のうちNAGISA_V3が勝った割合である。
- **Elo差**はNAGISA_V3を基準とし、正ならNAGISA_V3側、負なら対戦相手側が高い。
- Elo差の後ろには95%信頼区間を付記する。区間が広い結果は、数値の不確実性も含めて
  読む必要がある。

勝率は、次の式で計算する。

```text
勝率 = 勝数 / (勝数 + 敗数)
```

<a id="conditions"></a>

## 対局条件

各対局は初期8秒、1手ごとに0.08秒加算、1スレッド、Hash 64MB、Ponderなしで
実施した。中盤の互角局面集を先後入れ替えて使用し、定跡は使用していない。

### 有料AIとの比較

奏乗は有料で販売されており、水匠11は有料のFANBOXで配布されている。

<a id="tsec7-engine"></a>

#### 奏乗（TSEC7版）

奏乗は有料で販売されている将棋AIで、TSEC7で3位に入賞した。対局にはTSEC7版を
使用した。奏乗本体と探索部は同じ作者によるもので、今回の比較対象の中では唯一、
本家やねうら王の探索部を使用していない。

NAGISA_V3は全5,042局で2,049勝2,437敗556分、引き分けを除いた勝率は45.7%だった。
Elo差は−26.79 ± 8.86で、この対局条件ではNAGISA_V3が奏乗（TSEC7版）を下回った。

| 項目 | 内容 |
|---|---|
| 対戦相手 | 奏乗（TSEC7版） |
| 対局数 | 5,042局 |
| 成績 | 2,049勝 2,437敗 556分 |
| 勝率（引分除外） | 45.7% |
| Elo差 | −26.79 ± 8.86（95%信頼区間） |

<a id="suisho11-v940"></a>

#### 水匠11（v9.60git＋V9.40探索パラメータ）

この対戦相手は、本家やねうら王のv9.60git版ソースに水匠11の評価関数を組み合わせ、
有料のFANBOXで配布されたV9.40用の探索パラメータを焼き込んだ構成である。V9.40向けに
調整された探索パラメータを後のv9.60git版ソースへ適用しているため、この組み合わせに
対して最適とは限らない。この結果は、この特定の組み合わせとの比較を示すものであり、
水匠11本来の棋力全体を示すものではない。

NAGISA_V3は全5,158局で2,633勝2,112敗413分、引き分けを除いた勝率は55.5%だった。
Elo差は+35.21 ± 8.97で、この対局条件ではNAGISA_V3が水匠11を小幅ながら明確に
上回った。

| 項目 | 内容 |
|---|---|
| 対戦相手 | 水匠11 |
| エンジンソース | 本家やねうら王 v9.60git版 |
| 対戦相手の評価関数 | 水匠11 |
| 探索パラメータ | 有料のFANBOXで配布されたV9.40用パラメータ |
| 対局数 | 5,158局 |
| 成績 | 2,633勝 2,112敗 413分 |
| 勝率（引分除外） | 55.5% |
